---
title: Shapes
description: The AgentOS ontology — how every entity is described. Shapes declare what a record looks like so skills and apps agree on field names, types, and relations.
---

Shapes are how AgentOS describes what things *are*. A `message` has `from`, `to`, `body`, `sent_at`. A `book` has `title`, `author`, `isbn`. Skills write conformant dicts; apps read them; the engine is shape-agnostic.
