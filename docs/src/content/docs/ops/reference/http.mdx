http.request — the one stateless HTTP op.

The SDK's client module composes cookies/headers and calls this op;
the engine is pure transport. Handlers live in agentos-exec-executors.

## `http.request`

Stateless HTTP op. SDK callers use client.get/post/… — this op is their shared backend.

| Metadata | Value |
|---|---|
| **Capability** | `http` |
| **Flags** | `trace_span` |
| **Audit log** | `response/status` → `status`, `response/body_length` → `body_length`, `response/content_encoding` → `enc` |

### Request

| Field | Type | Required | Default | Description |
|---|---|---|---|---|
| `url` | `string` | **yes** |  | Absolute URL. |
| `method` | `string` | no | `'GET'` | HTTP method: GET, POST, PUT, DELETE, PATCH, HEAD. |
| `headers` | `map<string,string>` | no | `{}` | Request headers. Cookies ride here under "Cookie". |
| `params` | `map<string,json>` | no | `{}` | Query string parameters; appended to the URL. |
| `body` | `json?` | no |  | Raw request body — bytes or string, sent verbatim. |
| `http2` | `boolean` | no | `True` | Whether to enable HTTP/2. Set false for JA4-fingerprint hosts. |
| `timeout` | `number` | no | `30` | Request timeout in seconds. |

### Response

| Field | Type | Description |
|---|---|---|
| `status` | `u16` | HTTP status code. |
| `body` | `string?` | Response body decoded as UTF-8. |
| `body_bytes` | `string?` | Hex-encoded response body when non-UTF-8. |
| `headers` | `map<string,string>` | Response headers (lowercased keys). |
| `url` | `string?` | Final URL after the redirect chain. |
| `body_length` | `u64?` | Decoded body length in bytes. |
| `content_encoding` | `string?` | Content encoding (gzip, br, zstd, or none). |

### Notes

```text
Stateless HTTP op. Every verb-dispatched SDK method (`client.get`,
`client.post`, …) routes through this; skill code does not call
`http.request` directly. No cookie-jar state survives between calls —
the SDK's ambient Jar serializes cookies into the request's Cookie:
header and feeds Set-Cookie responses back into itself.

Args:
    url, method, headers, params, body, http2, timeout.

Returns:
    HttpResponse {status, body?, body_bytes?, headers, url, body_length, content_encoding}.
```
