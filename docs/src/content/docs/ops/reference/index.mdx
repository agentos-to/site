## Named record types

### `provider`

| Field | Type | Description |
|---|---|---|
| `skill_id` | `string` | Skill id that declared @provides(name, ...). |
| `via` | `string` | Operation name (via) on the skill that would be invoked. |
| `cred_state` | `string` | Credential state: not_required, present, or missing. |
| `urls` | `string[]?` | Optional URL patterns declared on the @provides entry. |
